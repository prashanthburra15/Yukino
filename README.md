# Yukino – Character-Based Conversational AI Dataset

## What is this project?

Yukino is a project where I am building a **Conversational AI dataset**, but unlike a typical chatbot dataset, this project focuses on **character-driven conversations**.

Instead of generating generic responses, the dataset is designed so that the AI responds with a **specific personality (Yukinoshita-style)**.

---

## What makes this different from a normal chatbot?

Most chatbot datasets:
- Give general-purpose replies  
- Focus only on correctness  

This project:
- Focuses on **how a specific character speaks**
- Maintains **tone, attitude, and personality**
- Generates **consistent character-based responses**

👉 In simple terms:  
This is not just a chatbot dataset — it is a dataset for building a **character AI**.

---

## How does this project work?

The dataset is built using context-response pairs.

Example:

{
  "context": "Hikigaya: Huh?",
  "response": "Yukinoshita: If you can have a conversation with a girl like me, you’ll be fine with just about anyone."
}

### Flow:

1. A context (input message) is defined  
2. A response is written in **Yukino’s personality style**  
3. Many such pairs are stored  
4. AI can learn:
   - conversation flow  
   - response generation  
   - personality consistency
